# 🍌 Nano Banana Studio

基于 Google Gemini API 的原生图片生成 Web 工具，支持文生图、图片编辑、多轮对话迭代，兼容官方 API 及第三方中转服务。

## 截图

> 打开 `index.html` 即可使用，无需安装任何依赖。

## 功能特性

### 图片生成
- **文生图**：输入提示词，一键生成高质量图片
- **图片编辑**：上传最多 14 张参考图片，配合提示词进行编辑、合成、风格迁移
- **多轮对话**：自动保留上下文，可持续迭代修改图片（"让光线更暖一些"、"改成西班牙语"…）

### 模型支持

| 模型 | API 标识 | 特点 |
|---|---|---|
| 🍌 **Nano Banana 2** | `gemini-3.1-flash-image-preview` | 速度与质量均衡，推荐首选 |
| 🍌 **Nano Banana Pro** | `gemini-3-pro-image-preview` | 专业素材、高保真文字渲染、4K 输出 |
| 🍌 **Nano Banana** | `gemini-2.5-flash-image` | 高速低延迟，适合批量场景 |

### 输出配置
- **宽高比**：14 种（1:1 / 16:9 / 9:16 / 4:3 / 21:9 / 1:4 / 8:1 等）
- **分辨率**：512 / 1K / 2K / 4K
- **思考等级**：`minimal`（低延迟）/ `high`（高质量）
- **可选项**：显示思考过程、仅输出图片、Google 搜索接地、Google 图片搜索接地

### 其他
- 点击图片放大灯箱预览
- 一键下载生成图片
- 将生成图片直接设为参考图（用于下一轮编辑）
- 拖拽上传参考图片
- API Key 及 Base URL 本地保存（localStorage），刷新不丢失

## 使用方法

### 1. 获取 API Key

- **官方**：前往 [Google AI Studio](https://aistudio.google.com/) 获取免费 API Key
- **第三方中转**：使用任意兼容 Gemini API 格式的服务（如 `https://newapi.aiopus.org`）

### 2. 打开页面

直接用浏览器打开 `index.html`，**无需服务器、无需安装**。

### 3. 配置

在顶部 Header 中填写：

| 字段 | 说明 |
|---|---|
| **Base URL** | 留空使用 Google 官方；第三方中转填入对应地址，如 `https://newapi.aiopus.org` |
| **API Key** | 你的 Gemini API Key 或第三方 Key |

点击「**保存**」后配置持久化，下次打开无需重填。

### 4. 生成图片

1. 在左侧「**提示词**」区域输入描述
2. 可选：上传参考图片（支持最多 14 张，拖拽或点击上传）
3. 按需调整宽高比、分辨率、思考等级等参数
4. 点击「**🍌 生成图片**」
5. 生成完成后，可在底部快速输入框继续下达修改指令

## 提示词技巧

- **描述场景而非关键词**：叙述性段落比关键词堆砌效果更好
- **指定摄影参数**：如"85mm 人像镜头"、"黄金时段光线"、"浅景深"
- **明确用途**：如"为高端护肤品牌设计徽标"比"设计徽标"效果更好
- **分步指令**：复杂场景可拆分成多步，逐步完善
- **迭代优化**：利用多轮对话小幅调整，"效果不错，但让颜色更饱和一些"

## 技术说明

- 纯前端单页应用，零依赖，零构建
- 直接调用 Gemini REST API（`/v1beta/models/{model}:generateContent`）
- 兼容返回 base64 `inlineData` 的官方格式及返回 Markdown 图片 URL 的第三方格式
- 对话历史保存在内存中，刷新页面会重置

## API 响应格式兼容

| 格式 | 说明 |
|---|---|
| `inlineData` (base64) | 官方 Gemini API 标准格式 |
| `![alt](https://...)` | 部分第三方中转 API 返回的 Markdown 图片链接，自动解析展示 |
